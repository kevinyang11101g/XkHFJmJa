# 项目简介 python75


## 技术栈

- **后端语言与框架**：Python 3.x，Django
- **数据处理**：Pandas
- **数据库**：MySQL
- **数据可视化**：Echarts
- **前端技术**：自适应设计，支持跨平台

## 功能模块

- **用户注册与登录**：允许用户创建账户，登录系统。
- **图书推荐**：根据用户历史行为和协同过滤算法，为用户提供个性化图书推荐。
- **热门书籍浏览**：展示当前热门的图书，供用户浏览。
- **数据可视化**：通过Echarts将书籍评分、用户偏好和市场趋势以图表形式展示。
- **社交推荐**：创新的社交推荐功能，增强用户间的互动。

## 系统角色

- **普通用户**：可注册、登录，查看推荐书籍，浏览热门书籍。
- **后台管理员**：负责管理用户信息和书籍数据。

## 运行环境

- **操作系统**：跨平台支持
- **依赖管理**：使用pip管理Python依赖
- **硬件要求**：常规服务器配置

## 部署步骤

1. 安装Python 3.x和MySQL数据库。
2. 使用pip安装项目所需依赖。
3. 配置数据库连接。
4. 部署后端服务。
5. 部署前端页面。
